A classic Snake game implemented in C# using Windows Forms with dynamic grid, score tracking, and smooth UI.

## Features
- Arrow keys to move the snake.
- Random food generation.
- Score and High Score tracking.
- Pause/Resume with `P`.
- Restart after Game Over with `Space` or Start button.

## Controls
| Key         | Action                        |
|------------|-------------------------------|
| Arrow Keys | Move the snake                |
| P          | Pause/Resume                  |
| Space      | Restart game after Game Over  |

## How to Run
1. Open the project in **Visual Studio**.
2. Build and run.
3. Click the canvas to focus.
4. Use arrow keys to play.

## Game Logic
- Snake moves each tick, grows when eating food.
- Collision with walls or self ends the game.
- UI updates score and overlays dynamically.
